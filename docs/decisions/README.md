## 决策记录规则

- 只有重大产品、架构、数据和安全决定才进入本目录。
- Agent 不得未经产品负责人确认创建重大产品决策。
- 新决策应说明背景、决定、原因、备选方案和重新评估条件。
- 新增、替代或废弃决策时，应同步更新本索引中的状态和备注。
