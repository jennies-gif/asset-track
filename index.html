    数据仅供记录和分析，不构成投资建议。

    Privacy Policy / 隐私政策

    资产轨迹 MVP 以本地保存为主。你录入的资产、交易、复盘和设置默认保存在当前浏览器的 localStorage 中。

    Terms / 使用条款

    本产品用于个人资产记录、组合分析和投资复盘。用户需要自行保证录入数据、价格、汇率、费用和备注的准确性。

    Risk Disclaimer / 风险提示

    资产轨迹不提供投资建议。页面展示的收益、风险、归因和复盘内容仅用于记录、核对和回顾。

    Data Backup Guide / 数据备份说明

    建议在集中录入、修改或导入前后导出 JSON 备份。

    Feedback / 用户反馈

    点击页脚“用户反馈”可以打开反馈模板。反馈时请不要发送完整资产备份文件、真实账户凭据、身份证明或其他敏感材料。

    Changelog / 更新日志

    已支持：本地资产记录、交易记录、收益概览、复盘、JSON 导入导出和 CSV 导出。

    确认导入备份数据？

    导入后会替换当前浏览器中的本地数据。建议你先导出当前数据备份，并确认导入文件来自你信任的来源。

    要记录这次操作吗？

    清仓后做一次简短复盘，后面回看决策会更清楚。

    发送试用反馈

    告诉我们哪里卡住、哪个数字不可信、哪个功能最有价值。请不要粘贴完整资产备份、真实账户凭据或身份证明。
